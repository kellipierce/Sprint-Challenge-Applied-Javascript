SELF-STUDY QUESTIONS 1. What is the DOM? Document Object Model - this model contains every element in order on the page 2. What is an event? Every user interaction with a site (i.e. a click, moving the mouse, scrolling the page, pressing a key on the keyboard) 3. What is an event listener? This allows us to choose specific elements to listen to specific events and fire a callback when that event occurs We can add event listeners for as many events as there are for each element 4. Why would we convert a NodeList into an Array? NodeLists are a browser API (they can access DOM elements) and not actually a JavaScript API 5. What is a component? A component is made of several parts: HTML, CSS, or JavaScript brought together for reuse in a website or application; they are the building blocks of any react app
